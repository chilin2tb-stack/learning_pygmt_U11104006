影片教學核心流程

1. The Canvas

一切從 fig = pygmt.Figure() 開始。這是 PyGMT 繪圖的核心物件，類似於畫布的概念。

2. Basemap & Projection

使用 fig.basemap() 設定地圖邊框。影片示範了從 Mercator (M) 到 Hammer (H) 投影的轉換。參數 region='g' 代表全球數據。

3. Remote Data

最強大的功能：fig.grdimage()。只需輸入 @earth_relief_20m，PyGMT 會自動從伺服器下載地形數據並繪製，無需手動處理檔案。
